How to Use It

$(document).KeyDownCombo({ 
    keys: [38, 38, 40, 40, 37, 39, 37, 39, "B", "A"]
});

API

Required

  • JQuery.KeyDownCombo.keys
    keycode or words(igrone case)

Options

  • delayTime
    Deytime between two key down time.
  • limitType
    [default: none, 1: only numbers, 2: only words, 3: Only numbers and words]
  • igrone
    keycode or words(igrone case)

Events

  • onTimeOutError
    When out key down delay time.
  • onKeyError
    When key down wrong key.
  • onSuccess
    When key down right key.
  • onComplete
    When combo is done.
